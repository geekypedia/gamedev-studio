# 📦 Game Dev Studio Installer (Ubuntu)

## Overview

Game Dev Studio Installer is a unified setup script for Ubuntu that transforms a fresh system into a complete **game development and creative production workstation**.

It installs and configures:

- 🎮 Game engines (Godot, GDevelop, ct.js, Ren’Py, Love2D, microStudio)
- 🌐 Web game stack (Node.js, React, Phaser, ExcaliburJS)
- 🎨 Art tools (GIMP, Krita, Inkscape, Piskel, Pixelorama, LibreSprite)
- 🎧 Audio/Video production suite (LMMS, Audacity, Ardour, Hydrogen, synth tools, Kdenlive, OBS Studio, VLC)
- 💻 Development tools (VS Code, code-server, Chrome, git)
- 💻 Productivity tools (Obsidian)
- 📦 Deployment tools (itch.io Butler)
- 🧠 Pipeline utilities (LDtk sync system)

This project is designed to give developers a **single-command setup for a complete indie game studio environment**.

---

## ⚡ One-Line Installation Command

```bash
curl -fsSL https://raw.githubusercontent.com/geekypedia/gamedev-studio/main/install.sh -o ~/gamedev-studio.sh && chmod +x ~/gamedev-studio.sh && sudo ~/gamedev-studio.sh
```
